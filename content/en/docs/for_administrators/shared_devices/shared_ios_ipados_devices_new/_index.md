This subsection documents one deployment only: **Shared iPad with MSAL**.

It covers the current setup that uses:

- **IDmelon Authenticator** in shared mode
- **Microsoft Authenticator** in shared mode
- **Microsoft Intune** for app deployment and managed app configuration
- **Microsoft Entra ID** for the MSAL app registration
- **Microsoft Enterprise SSO** for Microsoft app sign-in reuse
